Kim jestem?

Jestem młodym i ambitnym chłopakiem który chce coś w życiu osiągnąć. Aspiruję do bycia świetnym programistą i wybitnym pisarzem. Lubię podejmować wyzwania oraz uczyć się nowych rzeczy więc rzadko się nudzę. Pasjonuję się nie tylko programistyką ale też muzyką i pisarstwem. Wierzę w to, że ciężką pracą można osiągnąć wszystko!

Języki w których pracuję

JavaScript

JavaScript

JavaScript uczę się od dwóch lat. Jest to jeden z moich ulubionych języków przez to jak ciekawie można go wykorzystać chociażby do animacji.

c++

C++

C++ był jednym z pierwszych języków w których programowałem dlatego jestem z nim całkiem dobrze obeznany.

Python

Python

Python jest jednym z języków który ciekawi mnie najbardziej. Początkowo sprawiało mi problemy to jak prosty jest, jednak teraz gdy dostrzegam jego możliwości jestem podekscytowany.

Do czego dążę?

Moim nadrzędnym celem jest nie stanie w miejscu. Świat ciągle się rozwija więc nie mogę zostawać w tyle. Największe ambicję wierzę z programistyką oraz pisarstwem. W wolnym czasie lubię pisać kod oraz opowiadania.Mimo, że oba hobby nie są do siebie podobne to wierzę, że uda mi się odnieść sukces w obu tych dziedzinach!
